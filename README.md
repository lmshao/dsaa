## 排序算法

### 插入排序
把第二个数据作为参考值，把前面的值一次与参考值比较，大的话就挪到参考值位置，直到前面的某个值比此参考值小，就把参考值放在此位置。

### 冒泡排序
两层循环，里面的循环只比较相邻的两个元素，前者比后者大就交换位置，里面的循环每次都能保证最大的一个元素冒泡到最后面。然后进行多轮，这样就都是有序的了。

### 选择排序
选择第一个元素为基准元素，每次都选取基准元素后面的最小的一个元素与基准元素交换。基准元素每次都往后移。
